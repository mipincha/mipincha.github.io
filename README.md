# 🐜 Mi Pincha - Agencia de Empleos

**Conectamos talento con oportunidades**

Plataforma de empleo 100% gratuita, sin cloud US, construida sobre Codeberg/GitHub Pages.

## 🚀 Características

- ✅ **100% Estático**: Sin backend tradicional
- ✅ **Append-Only**: Todos los datos en JSON
- ✅ **Batching**: Envío eficiente cada 30s o 10 jobs
- ✅ **Sharding**: Datos organizados por fecha
- ✅ **Sin Tokens Expuestos**: Seguridad por diseño
- ✅ **PWA**: Instalable en móviles
- ✅ **Offline-First**: Funciona sin conexión
